# Screenshots

Place screenshots here using these exact filenames so the main `README.md` picks them up:

| Filename | What to capture | Suggested size |
|---|---|---|
| `hero.png` | The whole app — code on the left, diagram on the right. The "money shot." | 1600 × 900 |
| `editor.png` | Close-up of the Monaco code editor with Mermaid syntax highlighting. | 800 × 600 |
| `style-panel.png` | The Style popover open with the theme presets, color pickers, font dropdown. | 800 × 600 |
| `present-mode.png` | Present mode active, showing the current node and the breadcrumb path. | 800 × 600 |
| `export-dialog.png` | The Export dialog open with PNG/SVG/PDF selector and preview area. | 800 × 600 |
| `share.png` | Toast or visual showing the share link being copied. | 800 × 600 |
| `history.png` | The history sidebar with a few saved diagrams. | 800 × 600 |

**Tip:** PNG with a transparent or app-background border works best. Keep the file size under ~400 KB each (use [TinyPNG](https://tinypng.com) or similar if needed).
